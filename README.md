# YowPainter Frontend

Bienvenue sur le dépôt du frontend de **YowPainter**, une plateforme numérique dédiée à l'art et à la mise en valeur du travail des peintres. 

## Description de l'Application

YowPainter est une application web conçue comme une véritable galerie d'art numérique. Elle permet aux artistes d'exposer leurs œuvres de manière élégante, aux acheteurs de découvrir et d'acquérir de nouvelles toiles, et aux visiteurs d'explorer une collection d'art visuel. 

L'interface utilisateur a été spécialement pensée pour offrir une expérience "musée" premium :
- **Design Artistique** : L'interface simule une véritable toile de peinture avec des effets visuels soignés (bords de cadre, taches de peinture en SVG).
- **Parcours Utilisateur Immersif** : Des animations d'apparition fluides au défilement et des détails ergonomiques (comme un curseur personnalisé) plongent l'utilisateur dans une atmosphère créative et haut de gamme.
- **Performances** : Une navigation fluide et rapide permettant de mettre les œuvres au centre de l'attention.

## Stack Technique

Le projet repose sur des technologies web modernes et performantes :

- **Framework** : [Next.js](https://nextjs.org/) (Version 16, App Router)
- **Bibliothèque UI** : [React](https://react.dev/) (Version 19)
- **Langage** : [TypeScript](https://www.typescriptlang.org/) pour la robustesse et le typage statique
- **Styles** : [Tailwind CSS](https://tailwindcss.com/) (Version 4) pour un design système rapide, flexible et sur mesure
- **Animations** : Framer Motion (ou CSS natif) pour les animations "reveal" et les interactions dynamiques

## Lancement du Projet en Local

Pour faire tourner le projet sur votre machine, suivez ces étapes :

1. Installez les dépendances :
   ```bash
   npm install
   ```
2. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```
3. Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur pour voir le résultat.
